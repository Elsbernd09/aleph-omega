# Architecture

Project ℵω is organized into several major modules.

## `src/generative_axioms`

This module generates and evaluates toy axiom systems.

It will eventually include:

- axiom representation
- axiom libraries
- structural scoring
- contradiction-risk heuristics
- novelty scoring
- evolutionary search

## `src/toy_topoi`

This module defines simplified logical universes with different internal truth systems.

It will eventually include:

- classical logic
- intuitionistic logic
- paraconsistent logic
- many-valued logic
- modal logic
- fuzzy logic

## `src/bridges`

This module studies translations between formal universes.

It will eventually include:

- statement translation
- truth-value transport
- loss-of-meaning detection
- bridge scoring
- source and target universe comparisons

## `src/cognitive_morphism`

This module explores the translation of informal intuition into structured symbolic form.

It will eventually include:

- natural-language intuition parsing
- mathematical object detection
- symbolic statement construction
- Lean theorem skeleton generation

## `lean`

This folder contains Lean experiments.

The Lean files will begin with simple proposition and proof-object examples before moving into reflection and formalization experiments.

## `experiments`

This folder contains runnable experiments and research-style outputs.

## `docs`

This folder contains the written explanation of the project.
