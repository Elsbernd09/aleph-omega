# Functorial Semantics Examples

## Purpose

Phase 13 connects categorical bridge structure with satisfaction-based semantics.

The central question is:

> When does semantic preservation survive composition?

These examples show how bridge chains can preserve satisfaction or accumulate distortion.

---

## Example 1: Identity Transport

The identity bridge transports a truth assignment from a universe back to itself.

If a statement is true before identity transport, it remains true after identity transport.

This is the simplest semantically well-behaved case.
